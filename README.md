## ✨ 功能特性

- **智能填充**：页面加载完成后，自动检测评论表单并插入数据，省去手动填写步骤。
- **无缝集成**：与市面上 99% 的博客系统兼容，无需额外配置，轻松集成。
- **数据加密**：采用加密技术，确保用户数据的安全和隐私不泄露。
- **现代化界面**：基于 Material-UI 和 React 现代化 UI。

---

## 🚀 安装

1. 前往 Chrome 应用商店
2. 搜索并访问 <a href="https://chromewebstore.google.com/detail/eamchegekphehbmebccbapnihegngobm?utm_source=item-share-cb" target="_blank">EasyFill</a>。
3. 点击 **添加到浏览器** 按钮完成安装。
4. 安装成功后，浏览器工具栏将显示 EasyFill 图标。
